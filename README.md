# Data Visualization with the Altair library

## How to run
To run the analysis notebook
```bash
$ git clone https://github.com/tarujg/viz-altair.git
$ cd viz-altair/
$ python3 -m venv viz
$ source viz/bin/activate
$ pip install -r requirements.txt
```
